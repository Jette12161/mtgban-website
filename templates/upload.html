Instructions


  • Upload a file and let the Magic begin!
  • This tool can read data from a CSV file or from the first sheet of an Excel file.
  • The result will be a page containing Retail or Buylist prices from all the available stores on BAN (or from the user-selected ones) for each recognised card entry.
  • Note that buylist prices are available for our Legacy patrons only.
  • Data may be in any format, as long as the file contains a header in the first row with sensible indication on what the column contains (eg. card name and edition as a minimum).
  • For this reason, the most popular CSV files are supported out of the box, such as from TCG, Deckbox, or Cardsphere.
  • The file cannot be larger than 10MB, and it may contain up to 100 entries.
  • Multiple conditions and quantity fields are currently ignored.
  • Card recognition works best when data strictly follows Scryfall notation. If it fails, try finding your card there and tweaking your entry accordingly.
  • If anything doesn't work, post your file on Discord and we'll take a look.
